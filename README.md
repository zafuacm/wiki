# ZAFU ACM Wiki

校队自用 Wiki。

## Why

因为网上的资料过于杂乱，我希望能整理一些更有可读性的文章，让资料更有价值，让知识更有序。

也有很多大型项目在干同样的事情，如 [OI-Wiki](https://oi-wiki.org)，虽然文章质量很高但是并不容易读懂（我太菜了）。毕竟大学才开始接触编程，需要更低的门槛。

我更期望这里文档具有线性可读性，能够像一本书一样顺序的学习，因此我加入了 MdBook 支持，这也意味着表达形式会受到限制。
