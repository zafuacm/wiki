多项式牛顿迭代 ​

尝试不带 FFT/NTT 讲解，似乎真的能讲？

对于给定的 A(g)=0，假如我们已经求得 g0=gmodxn，那么可以倍增的求出 g1=gmodx2n。

对 A(g) 在 g=g0 处做泰勒展开

0=A(g)=A(g0)+A′(g0)1!(g−g0)+A″(g0)2!(g−g0)2+⋯

注意到 g−g0 前 n 项都为 0，那么

0≡A(g)≡A(g0)+A′(g0)(g−g0)(modx2n)

因此

g≡g0−A(g0)A′(g0)(modx2n)

更具体的，可以写做

g≡g0−A(g0)∂A′∂g(g0)(modx2n)

多项式 inv ​

不妨设 A(g)=f∗g−1=0，套公式有

g≡g0−f∗g0−1f≡g0−(f∗g0−1)∗g0≡2g0−f∗g02(modx2n)
cpp
Poly inv(int m) const {
    Poly x = {qpow(T[0])};
    for (int t = 2; t < m * 2; t *= 2) {
        x = x * 2 - (x * x) * cut(t);
        x.redeg(t);
    }
    return x.redeg(m);
}

Poly div(int m, const Poly& g) const {
    if (deg() == 0)
        return {};
    return (cut(m) * g.inv(m)).redeg(m);
}

多项式 exp ​

令 A(g)=ln⁡g−f=0，有

g≡g0−ln⁡g0−f1/g0≡g0∗(1−ln⁡g0+f)(modx2n)
cpp
Poly exp(int m) const {
    Poly x = {1};
    for (int t = 2; t < m * 2; t *= 2) {
        x = x * (Poly{1} - x.ln(t) + cut(t));
        x.redeg(t);
    }
    return x.redeg(m);
}

        x = x * (Poly{1} - x.ln(t) + cut(t));
        x.redeg(t);
    }
    return x.redeg(m);
}

多项式 sqrt ​

令 A(g)=g2−f=1，有

g≡g0−g02−f2g0≡12(g0+f/g0)(modx2n)
cpp
Poly sqrt(int m) const {
    Poly x = {1};
    for (int t = 2; t < m * 2; t *= 2) {
        x = (x + cut(t).div(t, x)) * qpow(2);
    }
    return x.redeg(m);
}
Poly sqrt(int m) const {
    Poly x = {1};
    for (int t = 2; t < m * 2; t *= 2) {
        x = (x + cut(t).div(t, x)) * qpow(2);
    }
    return x.redeg(m);
}

提示 ​

以上代码真的能跑，时间复杂度 O(nlog⁡n)，就是常数比较大。如果被卡常了，请去喷出题人。
